GOALS

How We Can Help Keep L.A Safe

By analyzing crime report data, we can come with ways to officers predict, prevent, and respond to crimes as efficiently as possible.

TRACKING CRIMES BY AREA

By tracking crime by area, we can gain a better understanding of what types of crimes may happen in what area.

CREATING A CRIME ACTIVITY MAP

A crime activity map may help officers track criminal activity in order to predict, prevent, and respond to future events.

DETERMINE MOST FREQUENTLY OCCURING CRIMES

Officers will be able to recognize the areas of criminal activity that need attention. This can also help the city to impliment precautions against certain criminal activity.

VICTIM DEMOGRAPHIC ANALYSIS

A victim demographic analysis will help officers identify vulnerable demographics. This is crucial to ensuring the saftey of L.A civillians.

AWS SERVICES

AWS Services Used to Bring This Together

EC2

A Linux virtual server that we are able to SSH into in order to make analyses and host this website.

S3

An object storage service used to store to store our data before it is to be retrieved by the Lambda function.

Redshift

A cloud-based data warehousing service for data storage, benefiting from scalability and managed services.

Lambda

A serverless computing function that pulls data from an API and uploads it to an S3.

Glue

A cloud-based ETL service used to automate the data transformation process.

IAM

An Identity and Access Management service used to control who can use the AWS resources attatched to the root user
